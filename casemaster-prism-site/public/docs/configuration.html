Configuration reference

Disable any enhancer at runtime via window.cmsUi. Disable a single element with data-cms-no-enhance.

Global config

<script>
  window.cmsUi = {
    tables:     { search: true, sort: true, csvExport: false },
    forms:      { tomSelect: true, flatpickr: true, selectThreshold: 8 },
    shortcuts:  true,
    command:    true,
    inlineEdit: false,
    confirm:    true,
    loading:    true,
    toasts:     true,
    modalLinks: true,
  };
</script>
<script type="module" src="/static/lib/prism/prism.js"></script>

Per-element opt-out

Skip a single element with data-cms-no-enhance:

<table class='table table-striped' data-cms-no-enhance>...</table>

<select class='form-control' data-cms-no-enhance>...</select>

<a href='/delete/42' data-cms-no-enhance>Skip enhancer</a>

Skip an entire subtree by adding the attribute on a wrapper:

<div data-cms-no-enhance>
  <!-- nothing inside this is enhanced -->
</div>
