Inline edit

Click a cell to edit in place. PATCHes back to a configured URL.

Usage

<td data-cms-inline-edit
    data-cms-edit-field="name"
    data-cms-edit-url="/api/inventory/42">
  Widget
</td>

<td data-cms-inline-edit
    data-cms-edit-field="qty"
    data-cms-edit-type="number"
    data-cms-edit-url="/api/inventory/42">
  120
</td>

Affordance

At rest, editable cells carry a dotted underline. On hover, a pencil icon (✎) appears alongside a primary-coloured outline. Click → an inline input replaces the cell content.

Commit / cancel

  • Enter or blur → PATCHes { field: value } to data-cms-edit-url; on success shows a 'Saved' toast.
  • Escape → reverts to the original value.
