Loading bar

Top-bar progress on link clicks and form submits. Plus busy-button state.

Triggers

  • Click on a same-origin <a> with no target="_blank".
  • Submit on any <form> not tagged data-cms-no-progress.
  • Manual: window.dispatchEvent(new CustomEvent('cms-progress-start')) / 'cms-progress-end'.

Busy buttons

On form submit, the first non-disabled <button type="submit"> gets aria-busy="true". The CSS replaces the button text with a spinner.

Manual API

import { startLoading, endLoading } from 'casemaster-prism';

startLoading();
await fetch('/api/heavy');
endLoading();
